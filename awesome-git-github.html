git 比较常用的命令

取消暂存区的操作:git reset HEAD [file]

取消对工作区文件的修改:git checkout -- [file]

给某个远程仓库加一个名字：git remote add pb git://github.com/paulboone/ticgit.git

查看远程仓库的信息:git remote show pb

查看当前HEAD指针指向的分支状态：git log --graph --pretty=oneline --abbrev-commit --decorate

将版本库更新到工作区:git checkout -- [filename]

git 其他命令使用

pull = fecth + merge

查看工作区文件和暂存区的差异:git diff

查看暂存区和上次提交时的差异:git diff --staged

已经跟踪的文件add和commit一起做：git commit -am "commit message"

删除暂存区和工作区的文件：git rm index.html

从暂存区删除 工作区的留着：git rm --cached index.html

重新修改提交说明 ，如果暂存区变了 会重新再次提交 但是是同一个提交:git commit --amend
